5. vw: Set the width of a box relative to the viewport width.

Create a box with a width of 50vw. What happens if you resize the browser window?

Box with vw units

As the viewport width increases or decreases, the width of the .box-vw will also increase or decrease respectively, always maintaining 50% of the viewport width.
For example:
If the viewport width is 1000px, the box will be 500px wide. If the viewport width is resized to 800px, the box will shrink to 400px wide. This behavior is ideal for responsive design, as the box adapts to different screen sizes without the need for media queries.

6. vh: Set the height of a box relative to the viewport height.

Create a box with a height of 25vh. How does the box’s height change when resizing the viewport vertically?

Box with vh units

As the height of the viewport increases or decreases, the height of the .box-vh will adjust dynamically, always maintaining 25% of the viewport's height.
For example:
If the viewport height is 800px, the box will be 200px tall (25% of 800px).
If the viewport height is resized to 600px, the box will shrink to 150px tall (25% of 600px).

7. vmin: Set a square’s size using the smaller dimension of the viewport.

Create a square div with width and height set to 20vmin. What happens if you change the browser’s orientation?

Square with vmin units

If the viewport's width is 1000px and the height is 800px, the smaller dimension is the height (800px), so the square will have a width and height of 160px (20% of 800px).
If the viewport's width is 800px and the height is 1000px, the smaller dimension is the width (800px), so the square will have a width and height of 160px (20% of 800px).

8. vmax: Set a square’s size using the larger dimension of the viewport.

Create a square div with width and height set to 20vmax. How does it compare to vmin when the viewport changes?

Square with vmax units

If the viewport's width is 1000px and the height is 800px, the larger dimension is the width (1000px), so the square will have a width and height of 200px (20% of 1000px).
If the viewport's width is 800px and the height is 1000px, the larger dimension is the height (1000px), so the square will have a width and height of 200px (20% of 1000px).

vmax makes the square's size based on the larger dimension of the viewport (either width or height).
vmin makes the square's size based on the smaller dimension of the viewport.
